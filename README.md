# Meeting Scheduler

This project is a simple meeting scheduler built using React.
Users can select a date, choose a time slot, enter their details, and confirm a meeting.

## Features

* Calendar date selection
* 15-minute meeting time slots
* Booking form (First Name, Surname, Email)
* Confirmation screen with meeting details
* Simulated email notification
* Responsive design for desktop and mobile

## Setup Instructions

Clone the repository:

git clone https://github.com/vinutha18-m/meeting-scheduler.git

Move into the project directory:

cd meeting-scheduler

Install dependencies:

npm install

Run the application:

npm start

The app will start at:

http://localhost:3000

## Notes

Email notifications are simulated as allowed in the assignment instructions.
